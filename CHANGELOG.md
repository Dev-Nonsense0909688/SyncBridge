# Changelog

## [0.1.0] - Initial Release

### Added

* UDP-based sync between peers (basic, not fully reliable)
* Memory allocator using mmap
* CLI **Daemon** commands: `daemon start`, `daemon stop`, `daemon status`
* Daemon system (tunnel + memory process)
* Cross-platform support (Windows and Linux)
* Basic registry system using fixed offsets
* Binary serialization using struct
